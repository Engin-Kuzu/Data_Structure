# Data Structure Implementations in C

This repository contains fundamental **Data Structure** implementations written in C. Each project is organized into its own directory with detailed documentation on its specific logic and functions.

---

## Overview of Implementations

This collection focuses on dynamic memory management and pointer manipulation to build efficient data storage and retrieval systems.

---

## 1. Linked List

A dynamic data structure consisting of nodes where each node contains data and a pointer to the next node.

*   **Key Features:** Dynamic sizing, efficient insertion, and deletion.
*   **Project Includes:** Basic node creation and a product management system sorted by quantity.

---

## 2. Queue

A linear data structure that follows the **FIFO (First In, First Out)** principle.

*   **Key Features:** Elements are added at the rear and removed from the front.
*   **Operations:** `enQueue`, `deQueue`, `peek`, and `display`.

---

## 3. Stack

A linear data structure that follows the **LIFO (Last In, First Out)** principle.

*   **Key Features:** All operations (insertion and removal) are performed from a single end called the top.
*   **Operations:** `push`, `pop`, `peek`, and `display`.

---

##  Tech

*   **Language:** C
*   **Concepts:** Dynamic Memory (malloc), Pointers, Linked Lists, Stack/Queue Logic.
